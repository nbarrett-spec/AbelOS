# Element: scope

What Abel actually supplies — the doors, trim, hardware ladder broken
out by tier so the builder can see exactly what they'd be specifying.

## Required content

Three sub-sections, in this order:

**Doors** — by manufacturer, with one tier per row.
- Masonite: Heritage (entry-level), Lincoln Park, Cheyenne, Riverside,
  Logan
- Therma-Tru: Smooth-Star (good), Fiber-Classic (better), Classic-
  Craft (best)
- ProVia: Embarq, Heritage, Signet
- Plastpro: standard fiberglass
- JELD-WEN: Smooth-Pro, Architectural
- Codel: Premium fiberglass, custom mahogany

**Trim** — primed pine (standard), MDF (cost-down), poplar paint-grade
(premium), solid stain-grade (custom).

**Hardware** — Builder-grade (Kwikset Halifax/Tustin), Mid (Schlage
F-Series), Premium (Emtek brushed brass / matte black, Baldwin).

## Layout & visual

- HERITAGE: prose-led with a small icon next to each manufacturer
  name. More photo-block real estate. One species detail per
  manufacturer.
- EXECUTIVE: tight three-column table per sub-section. Lead-time
  column on right.
- BUILDER_FIELD: flat single table with everything in one grid. PMs
  scan, they don't read.
