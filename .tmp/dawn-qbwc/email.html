Connecting QuickBooks to Aegis

Hi Dawn,

Quick one — we're connecting QuickBooks Desktop to Aegis so AR/AP, invoices, and bills flow into the dashboard automatically. You don't have to change how you work; this just listens.

Setup is one-time, ~10 minutes. Tools you'll install are free and from Intuit.

What you'll do

  1. Install QuickBooks Web Connector (free Intuit tool) — download here. Run installer; takes 1 min.
  2. Open QuickBooks with the Abel Doors and Trim company file. (QB needs to stay open during sync — it can be in the background.)
  3. Open the Web Connector from your Start menu.
  4. Click "Add an application" and browse to the attached file: abel-aegis.qwc
  5. QuickBooks will pop a permission box. Choose "Yes, whenever this QuickBooks company file is open" and click Continue.
  6. Back in Web Connector, paste this password and click Save:
    2ulPK_7z_uA5uG10iBjXME2ZOmD6qDIP
  7. Check the box next to "Abel Aegis Sync" and click Update Selected. First sync runs 5–15 min — let it finish.
  8. That's it. After that, sync runs automatically every 60 minutes whenever QB is open.

If something goes sideways

  • "Cannot connect to web service" — make sure QuickBooks is open and the company file is loaded.
  • "Application requires single-user mode" — switch QB to single-user (File → Switch to Single-user Mode), run the sync, then switch back.
  • Password rejected — copy/paste it exactly; it's case-sensitive. Don't type it.
  • Anything else weird — text Nate, he'll ping me.

Once it's running, you'll see a "QB · Last sync" indicator in Aegis showing it's healthy. The cash position, AR aging, and AP aging on the Finance dashboard will start updating from QB directly.
